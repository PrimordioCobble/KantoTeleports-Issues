# KantoTeleports - Issue Tracker

This repository is used only for bug reports, compatibility issues, and support requests related to the KantoTeleports Minecraft mod.

The source code is currently private and is not distributed publicly.

## Before opening an issue, please include:

- Minecraft version
- Fabric Loader version
- Fabric API version
- KantoTeleports version
- Java version
- Jade version, if installed
- Whether the issue happens in singleplayer, multiplayer, or both
- Whether the mod is installed on both client and server
- Crash report or `latest.log`, if applicable
- Steps to reproduce the issue

## For teleport block issues, please also include:

- Teleport block coordinates
- Target coordinates
- Sound ID configured on the block, if relevant
- Required item configured on the block, if relevant
- Whether the teleport was configured with the Teleport Setter item or commands

## For Room Session issues, please also include:

- Session ID
- Room ID
- The relevant part of `config/kantoteleports/room_sessions.json`
- Whether `/kantoteleports roomsession enabled true` is active
- How many players were inside the configured room area when the issue happened

## Official pages

Downloads, changelogs, and mod information are available on the official CurseForge and Modrinth pages.
